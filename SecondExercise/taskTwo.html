HTML Semantic tags

  1. <header> can contain
    • different titles(h1, h2, h3, h4, h5, h6)
    • navigation
      • <li>(list item)
      • <a>(hyper link)
  2. <main> can contain
    • different titles(h1, h2, h3, h4, h5, h6)
    • aside
      • <img>(images)
      • <section>(sections)
      • <article>(articles)
  3. <footer> can contain
    • navigation links (nav, a)
    • copyright symbols and information
